# McMenu

### Project setup
```
You need to install mongodb server before this

git clone git@github.com:dmytsuu/McMenu.git
OR
git clone https://github.com/dmytsuu/McMenu.git

cd McMenu

bundle install
yarn install
rails db:create
rails db:seed

foreman start
OR (in different tabs)
rails s
./bin/webpack-dev-server
```
